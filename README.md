# Battleships

* This is a 3-D game consisting of ships in an ocean.

--- 

## Objectives-

- Fire cannon balls to sink enemy ships.
- Collect treasures!
- Dodge from enemy cannons.

---

## controls-

* **w-** move forward
* **s-** move backward
* **a-** turn left
* **d-** turn right
* **x-** shoot cannon balls
* **c-** change camera view
